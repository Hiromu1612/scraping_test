## サービス名・概要
### 【ネットショップのスクレイピングアプリ】
 
・Amazonや楽天市場、PayPayフリマなどのショッピングサイトの情報をスクレイピング

・商品名・価格・送料・商品URLを表に一覧化する

・価格順・ポイント順に並び替える機能

・直近1か月・3か月・1年・2年の価格推移を表示する機能

URL: https://scrapingapp-3b15c3abeea4.herokuapp.com/
